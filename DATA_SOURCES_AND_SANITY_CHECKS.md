### 1.2 分數計算方法(可完全重現)

每個子訊號用**分段線性映射** `_scale(value, anchors)` 把原始指標轉為 0–100 分,錨點寫死在 `aibubble/fetcher.py`;子分數做算術平均;
層級總分 = Σ(子分數 × 權重)。**市場層權重**:CapEx 20%、估值 20%、動能 15%、基建溢價 15%、信用投機 15%、集中度 15%。**前瞻層**:GPU 租金 25%、剪刀差 20%、DC 壓力 20%、循環交易 15%、徵才 10%、收入缺口 10%。

### 1.3 例算(2026-06-10 快照,已驗證)

**例 1:CapEx 子分數 = 89.1 怎麼來**
原始輸入(Yahoo 季度現金流量表加總):五巨頭最新季 CapEx 合計 $148.4B、年增 **+90.81%**、CapEx/營運現金流 **94.0%**、CapEx/營收 **33.1%**。
- 年增率錨點 `[(0,15),(20,35),(40,60),(60,80),(90,95)]`:90.81 ≥ 90 → **95**
- /OCF 錨點 `[(30,15),(50,40),(70,70),(85,90),(100,100)]`:94 介於 85–100 → 90 + (94−85)/(100−85)×10 = **96.0**
- /營收錨點 `[(10,15),(20,45),(30,70),(40,90)]`:33.1 介於 30–40 → 70 + 3.1/10×20 = **76.2**
- 子分數 = (95 + 96.0 + 76.2) / 3 = **89.1** ✓(與頁面一致)

**例 2:市場層總分 = 64.5 怎麼來**
89.1×0.20 + 73.2×0.20 + 89.5×0.15 + 38.1×0.15 + 33.0×0.15 + 53.0×0.15
= 17.82 + 14.64 + 13.425 + 5.715 + 4.95 + 7.95 = **64.5** ✓

**例 3:前瞻層總分 = 51.9 怎麼來**
48.5×0.25 + 8.0×0.20 + 62.9×0.20 + 95.0×0.15 + 17.7×0.10 + 96.0×0.10
= 12.125 + 1.6 + 12.58 + 14.25 + 1.77 + 9.6 = **51.9** ✓

**例 4:行情漲跌幅口徑**
1日/1週(5 交易日)/1月(21)/6月(126)/1年 = 對應交易日前的調整後收盤價相比;52 週高 = 一年窗最高收盤。任何一檔都可以打開 Yahoo Finance 圖表手動對。

### 1.4 已知限制(對老闆要先講清楚的)

- Yahoo Finance 為非官方彙總,PE/PS 等欄位口徑與公司財報可能有差,**僅供橫向比較**;各巨頭財年季度截止日不同,「最新季」非同一日曆季。
- npm 下載含 CI/CD 流量 → **看趨勢不看絕對值**。
- GPU 回本帶 $1.3–1.9/hr 是**假設**(採購 $25–30k、5 年折舊、含電力機房、70% 利用率),方法寫明、可調。
- 循環交易金額為公開報導**承諾級別約數**,非實際撥付。
- 無 API key 時新聞分類為關鍵詞規則,精度低於 Claude 分類(頁面會標示分類器)。
- 分數高低代表「偏離歷史常態的程度」,**不是漲跌預測**。

## 二、US Economic Monitor(美國經濟指標)

### 2.1 資料來源

**單一來源:FRED(St. Louis Fed 官方 API)**,38 個指標(免費 API key)。完整序列清單:

| 類別 | FRED 序列 |
|---|---|
| 通膨 | CPIAUCSL(CPI)、CPILFESL(核心 CPI)、PCEPILFE(核心 PCE)、PPIACO(PPI)、MICH(密大通膨預期) |
| 就業 | PAYEMS(非農)、UNRATE(失業率)、ICSA(初領失業金)、CIVPART(參與率)、U6RATE、JTSJOL(JOLTS)、CES0500000003(時薪) |
| 成長 | GDPC1(實質 GDP)、RSXFS(零售)、INDPRO(工業生產)、PCE(個人消費)、CFNAI(芝加哥聯儲活動指數)、TCU(產能利用率) |
| 房市 | HOUST(開工)、PERMIT(建照)、EXHOSLUSM495S(成屋銷售)、MORTGAGE30US(30 年房貸)、CSUSHPISA(Case-Shiller) |
| 製造業 | MANEMP(製造業就業)、DGORDER(耐久財訂單)、IPMAN(製造業生產)、UMCSENT(密大消費信心) |
| 金融市場 | FEDFUNDS(聯邦基金利率)、T10Y2Y、T10Y3M(殖利率曲線)、DGS10、DGS2、VIXCLS、BAMLH0A0HYM2(HY OAS)、M2SL |
| 貿易與國際 | DTWEXBGS(美元指數)、DCOILWTICO(WTI)、BOPGSTB(貿易餘額)、IR(進口物價) |

(註:原 USSLIND 費城聯儲領先指數已於 2020 年停更,已剔除並以 CFNAI 取代。)

### 2.2 變化計算規則

每個指標存原始序列;「週/月/季/年變化」= 最新值 vs **距今 7/30/91/365 天最近的一筆觀測**(容差 = max(天數×0.4, 10 天))。月頻序列沒有 1 週前的觀測 → 1w 顯示「—」,這是設計而非錯誤。

### 2.3 例算(2026-06-09 快照,已驗證)

**例 1:CPI(月頻,format=index)**
最新 2026-04-01 = 332.407。
- 1月變化:ref 2026-03-01 = 330.293 → (332.407−330.293)/330.293 = **+0.64%** ✓
- 1季:ref 2026-01-01 = 326.588 → **+1.78%** ✓
- 1年:ref 2025-04-01 = 320.302 → (332.407/320.302−1) = **+3.78%** ✓(這就是新聞講的「CPI 年增率」)
- 1週:**—**(月頻無週度觀測,正確行為)
可直接在 FRED 網站搜 CPIAUCSL 對原始值。

**例 2:初領失業金 ICSA(週頻,format=count)**
最新 2026-05-30 = 225,000(顯示 225K)。
- 1週:ref 2026-05-23 = 212,000 → Δ +13,000 = **+6.13%** ✓
- 1月:ref 2026-05-02 = 199,000 → **+13.07%** ✓
- 1年:ref 2025-05 ≈ 244,000 → **−7.79%**(一年前更高)✓

### 2.4 已知限制

- FRED 數據會**回溯修正**(尤其非農、GDP),平台顯示的是抓取當下版本。
- 已知顯示問題待修:GDP/PCE/M2 在 FRED 原生單位是 Billions,顯示為「$24.15B」實為「$24.15T」(格式器假設 millions;RSXFS、貿易餘額正確)。**對外引用這三個指標時注意單位。**
- Render 免費方案冷啟動會還原成 repo 內 seed 資料,每週三排程/cron 會補抓最新。

## 三、Rival Radar(競爭者與客戶流向)

### 3.1 雙層資料架構

**A. 策展情報層**(`intel_kb.json`,情報基準日 2026-06-11)— 由 41 個 AI 研究 agent 產出:
6 個競爭者檔案 agent + 3 個客戶流向獵手 + 市佔數據 agent + 一手來源 agent,**每條事件再配 1 個獨立對抗式查證 agent、每份檔案配 1 個審查 agent**,最後由合成 agent 產出 CEO 摘要。每條宣稱都附真實 URL。

| 來源類型 | 例 | 證據層級 |
|---|---|---|
| 官方申報/法說會 | SEC 10-K/20-F/8-K/6-K、法說會逐字稿、官方新聞稿、韓國交易所重訊 | **T1 確證** |
| 權威量化 | TrendForce 季度代工市佔、Counterpoint、公司財報數字、主流財經媒體(Reuters/Bloomberg/Nikkei)的硬數字 | **T2 量化** |
| 供應鏈旁證 | DigiTimes、ETNews、Wccftech 等引述匿名消息的報導 | **T3 旁證(=傳聞,明確標示)** |

內容規模:6 檔案、**23 條客戶流向事件**(流失 9 / 台積電贏單 7 / 雙供應 3 / 評估中 3 / 反轉 1)、5 季 TrendForce 市佔(2024Q4–2025Q4)、12 列 capex、12 條法說會/申報原文引述。23 條事件逐條查證,**平均查證信心 0.80,其中 9 條的狀態被查證後修正**(例如把媒體寫成「已完成」的事項改回「進行中」)。

**B. 即時層**(按「更新」鈕重抓)— yfinance 10 檔競爭者行情 + Google News RSS 9 路雙語新聞(關鍵詞自動標「轉單/贏單/雙供應」訊號)。

### 3.2 例算(已驗證)

**例 1:台積電市佔 70.4%(2025Q4)從哪來**
TrendForce 2026-03-12 新聞稿(平台內可點擊):`trendforce.com/presscenter/news/20260312-12965.html`。同季 Samsung 7.1%、SMIC 5.2%、UMC 4.2%。市佔走勢圖五個季度各自附來源 URL。

**例 2:綜合威脅分數 = 五維加權,Intel 6.6 怎麼來**
權重:技術 0.28、客戶動能 0.28、產能 0.16、地緣 0.16、動能 0.12。
Intel 五維 = 技術 7、產能 5、客戶 5、地緣 9、動能 8:
7×0.28 + 5×0.28 + 5×0.16 + 9×0.16 + 8×0.12 = 1.96+1.40+0.80+1.44+0.96 = **6.56 → 6.6** ✓
(六家全部以同一公式驗算吻合;五維各自的評分理由寫在威脅儀表板卡片上。)

**例 3:一條事件的查證鏈長什麼樣(特斯拉 AI6 → 三星)**
- 原始狀態:confirmed / T2,來源 Bloomberg、CNN(2025-07)
- 獨立查證 agent 結論(信心 0.85):核心事件成立且有 **T1**(三星 2025-07-26 韓國交易所重訊揭露 $16.5B 合約至 2033);**修正**:設備進機 2026-04-24 才開始(非「已完成」)、AI6 時程因三星 2 奈米良率推遲約 6 個月。
- 頁面上點開該事件即可看到查證結論、修正說明與補充來源——**這就是「比新聞深一層」的機制**。

**例 4:即時行情怎麼算**
TSM 報價 $408.75、漲跌 +0.41% =(現價 − 前收)/前收;YTD +34.3% = 今年首個交易日收盤至今。皆可在 Yahoo Finance 直接對。

### 3.3 已知限制

- 策展層是**時點快照**(頁面標示情報基準日),不會隨「更新」鈕變動;更新鈕只重抓行情與新聞。
- T3 = 傳聞,平台刻意不和確證混排;若老闆看到 Apple/Qualcomm/AMD 等項目,請注意全部標示 rumored/evaluating。
- TrendForce 數字是其新聞稿口徑(T2),非審計數字。
- 營收影響(impact)為 1–5 級**評估分**,非精確金額。

## 四、三平台共同原則

1. **無黑箱**:所有映射錨點、權重、公式寫死在程式碼,任何人可重現本文件的每一筆例算。
2. **證據可點擊**:每條質性宣稱附原始 URL;量化數據附官方 API 序列名。
3. **AI 生成內容一律標示**(執行摘要、新聞分類),且有規則引擎 fallback。
4. **傳聞與確證分離**(T1/T2/T3 證據分級)。
5. 皆為**研究參考,非投資建議**;分數衡量偏離常態的程度,不是預測。
